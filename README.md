# Course Browser React

React + Vite + TypeScript ile yapılmış kurs keşif uygulaması.

## Özellikler
- Kurs listesi ve kart görünümü
- Kategori filtresi (Frontend, Backend, Fullstack, Mobile)
- Seviye filtresi (Başlangıç, Orta, İleri)
- Arama (kurs adı ve eğitmen)
- Sıralama (fiyat, puan)
- Kurs detay sayfası
- Hover animasyonları
- Loading state

## Kullanılan Teknolojiler
- React 18
- TypeScript
- Vite
- Custom Hook (useCourses)

## Kurulum
git clone https://github.com/EmreGokcek/course-browser-react.git
cd course-browser-react
npm install
npm run dev
